# Fridge Finder — Investor Pitch Deck

Interactive slideshow pitch deck for **Fridge Finder**: snap a photo of your fridge, get meal suggestions and a smart shopping list.

## Open the deck

Open `index.html` in a browser, or serve locally:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Navigation

- Arrow keys / Space / Page Up–Down
- On-screen ← → controls
- Progress dots
- Swipe on mobile
